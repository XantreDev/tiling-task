### Tiling challenge

Задачка с Yandex Cup 2025 на знание чистого HTML и CSS.

Есть доска с клетками
![alt](./Pasted_image_20251107225017.png)

Каждая клетка может быть закрашена или не закрашена. Если клетка соприкасется с другой закрашенной клеткой по диагонали или вертикали, то их надо соединить *ромбиком*, как на скринах.
![alt](./Pasted_image_20251107220645.png)
![alt](./Pasted_image_20251107221537.png)

![alt](./Pasted_image_20251107221854.png)
![alt](./Pasted_image_20251107221908.png)
![alt](./Pasted_image_20251107221924.png)
![alt](./Pasted_image_20251107221934.png)

Ваш код вставляется в каждую клетку поля и на родительском элементе проставляются состояния текущего элемента и соседних, за это отвечает набор классов
- `tl` - если клетка сверху слева закрашена
- `tr` - если клетка сверху закрашена
- `bl` - если клетка слева закрашена
- `br` - если текущая клетка закрашена
![alt]("./Pasted_image_20251107225537.png")

Код можно писать только внутри template-а c id="cell". На этом всё - можно открыть [файл-песочницу](./tiling-task.html) и начать веселиться
